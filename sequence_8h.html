struct geneie_sequence geneie_sequence_alloc ( ssize_t  length)

Allocates uninitialized memory for a geneie_sequence object and initializes its length member.

The resulting object can store length acid codes.

This object must be passed to geneie_sequence_free() when finished.

Parameters
lengthThe amount of codes to allocate memory for.
Returns
A pointer to a newly-allocated object, or NULL if allocation failed.
